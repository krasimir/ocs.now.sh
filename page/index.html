Octomments

Using GitHub issues as a comment plugin.

Star

Documentation · Author

Quick start. Add the following tag on the place where you want the comments to appear:

<div id="comments"></div>

In your page's <head> tag:

<link href="https://unpkg.com/octomments/build/ocs-ui.min.css" rel="stylesheet">

Then drop the following code at the end of the page.

<script src="https://unpkg.com/octomments/build/ocs.min.js"></script>
<script>
  Octomments({
    github: {
      owner: 'krasimir',
      repo: 'octomments',
    },
    number: 1,
    renderer: [OctommentsRenderer, '#comments']
  }).init();
<script>
