Octomments

Using GitHub issues as a comment plugin.

Star

Packages

  • Client (required) - you need this to fetch the comments from GitHub
  • Renderer (optional) - provides a default UI.
  • Server (optional) - without this your users will have to go to github.com to add new comments.

Assets

  • ocs-ui.min.css - CSS styles used by the default renderer
  • ocs.min.js - ocs-core.min.js + ocs-ui.min.js bundled in one file.
  • ocs-core.min.js - core client
  • ocs-ui.min.js - default renderer

Setup

<div id="comments"></div>

<link href="octomments-renderer.min.css" rel="stylesheet">
<script src="octomments-pack.min.js"></script>
<script>
  Octomments({
    github: {
      owner: 'krasimir',
      repo: 'octomments',
    },
    number: 1,
    renderer: [OctommentsRenderer, '#comments']
  }).init();
<script>
