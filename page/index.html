Octomments

Using GitHub issues as a comment plugin for your web site.

Star 

<div id="comments"></div>

<link href="octomments-renderer.min.css" rel="stylesheet">
<script src="octomments-pack.min.js"></script>
<script>
  Octomments({
    github: {
      owner: 'krasimir',
      repo: 'octomments',
    },
    number: 1,
    renderer: [OctommentsRenderer, '#comments']
  }).init();
<script>
Documentation:
  • Client (required) - you need this to fetch the comments from GitHub
  • Renderer (optional) - provides a default UI.
  • Server (optional) - without this your users will have to go to github.com to add new comments.
Octomments is an open source project and if you feel that something is wrong please consider opening a PR here.
